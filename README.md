# 💻 C# — Estudos e Programação Orientada a Objetos

Este repositório reúne meus estudos em **C#** com foco em **Programação Orientada a Objetos (POO)**, desenvolvidos a partir de um curso estruturado, mas organizados e implementados com minha própria abordagem.

---

## 🚀 Sobre

Durante o aprendizado, sigo como base um curso de POO em C#, porém não me limito à estrutura proposta.

Meu objetivo é:

* Adaptar os conteúdos à minha forma de entender
* Reorganizar os exemplos
* Criar variações e testes próprios
* Consolidar o conhecimento na prática

---

## 🧠 Abordagem de aprendizado

Em vez de apenas replicar o conteúdo, busco:

* Entender o funcionamento de cada conceito
* Reescrever códigos com minha própria lógica
* Testar diferentes soluções para o mesmo problema
* Organizar o código de forma clara e reutilizável

---

## 📂 Estrutura do repositório

O projeto está organizado para refletir a evolução dos estudos e facilitar a navegação:

```
📁 classes/        → definições e estruturas de classes  
📁 funcoes/        → métodos e comportamentos  
📁 poo/            → conceitos de orientação a objetos  
📁 projetos/       → aplicações práticas  
📁 exercicios/     → práticas e desafios  
```
